# Prime Exports & Trading Website

A modern, responsive, and animated static website for **Prime Exports & Trading (OPC) Pvt. Ltd.**, a rapidly growing merchant exporter in the pharmaceutical and healthcare sector based in Patiala, India.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS for custom, premium styling (Grid, Flexbox, custom variables, responsive design).
- **JavaScript**: Mobile navigation logic and initialization.
- **AOS (Animate On Scroll)**: For smooth fade, slide, and zoom animations as the user scrolls.
- **FontAwesome**: Icons.
- **Google Fonts**: Inter font family for modern typography.

## How to View Locally

Because this is a completely static website with no build step required, viewing it is incredibly simple:

1. Clone or download this repository to your local machine.
2. Open the folder `prime exports`.
3. Double-click the `index.html` file to open it directly in your preferred web browser (Chrome, Edge, Firefox, etc.).

## How to Deploy to GitHub Pages

This project is perfectly optimized for direct hosting on GitHub Pages.

1. Initialize a Git repository and commit all files in this directory.
2. Push your repository to GitHub.
3. In your GitHub repository, go to **Settings** > **Pages** (under the "Code and automation" section).
4. Under "Build and deployment", set the **Source** to "Deploy from a branch".
5. Set the **Branch** to `main` (or `master`) and the folder to `/ (root)`.
6. Click **Save**.
7. In a few minutes, GitHub will provide you with a live URL where your website is securely hosted for free.
